# random-str-msg-helio

## Description:

With this package you can see several conjugations in "Spanish" of the word verga in Maracaibo - Venezuela

## Install

```npm
npm install -g random-str-msg-helio
```

# Usage

```bash
 random-str-msg-helio
```
